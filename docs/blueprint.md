# **App Name**: MediSense

## Core Features:

- Patient Feedback Submission: Allows patients to submit their satisfaction rating, suggestions for improvement, and messages for staff via a user-friendly form.
- Feedback Data Storage: Securely stores all submitted patient feedback data, including ratings, text inputs, and submission timestamps, using Firestore.
- Satisfaction Rating: Patients can select a satisfaction level using a clear star rating system.
- Form Validation & Confirmation: Ensures essential fields are completed before submission and displays a clear success message upon completion.
- Administrative Feedback Viewer: Provides clinic staff with a basic interface to view and sort all collected patient feedback.
- AI Feedback Insights Tool: Uses AI to analyze and summarize text-based feedback, identifying key themes, sentiment, and urgent patient concerns for staff.

## Style Guidelines:

- A light color scheme based on a 'clean medical' feel. The primary color is a calming teal (#29A099). The background color is a very light, almost white, with a hint of the primary hue (#F2F7F7). A vibrant accent green (#4ED88C) is used to draw attention to interactive elements.
- The font 'Inter' (sans-serif) is used throughout for its modern, neutral, and highly legible appearance, suitable for both headlines and body text in a medical context.
- Utilize simple, line-based SVG icons related to healthcare, feedback, and submission to maintain a clean and professional aesthetic. Star icons are used for rating, consistent with the provided design.
- A minimalist, single-column layout centered on the page, with content confined to a comfortable reading width, optimized for responsiveness across various screen sizes with a mobile-first approach.
- Subtle micro-interactions on interactive elements such as buttons and stars, featuring slight transforms and color transitions on hover and click, to provide clear visual feedback without distraction.
